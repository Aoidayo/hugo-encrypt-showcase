# Hugo Encrypt Showcase

This repository is a minimal Hugo site using the Stack theme.

It is pinned to `github.com/CaiJimmy/hugo-theme-stack/v3 v3.34.2`, which is compatible with Hugo `0.143.1`.
This local setup also includes a small CSS override so the site can build with the non-extended Hugo binary at `/Users/aoi/software/env/hugo_0.143.1_darwin-universal/hugo`.

```sh
/Users/aoi/software/env/hugo_0.143.1_darwin-universal/hugo server -p 1314
```
